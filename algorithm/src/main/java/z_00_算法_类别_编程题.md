#流程模板
问题拆分:子问题(子模块)->子模块拼接
```asp
1.举例读懂题目含义
2.列举测试用例(测试驱动开发)
3.梳理题目合法条件,实现子模块
4.第一轮写代码,写注释
5.主线流程拼接子模块
6.测试用例case验证,字符串/整型,合法条件
```
