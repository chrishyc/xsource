```
http://c.n/X0BTU8
```

## 2.方案设计
查询时,获取源地址
a.在redis缓存中查找是否存在,redis保存长地址的hash还是源地址?
b.查数据库是否存在源地址,根据长地址对应的hash值查询
c.将源地址的(hash+id)转换为base62,插入数据库,并加入redis缓存
