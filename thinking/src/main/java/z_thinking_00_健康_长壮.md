# 最佳训练
## 机械张力
当前极限重量,1RM
## 代谢压力
极限次数6-12次,超过12次作用不大,6RM-12RM
起始负荷10次,调整负荷15次(当前重要能到15次时,增加重量)
## 超量恢复
状态比较好进行第二次训练
## 训练速度
60秒休息时间
## 训练顺序
大肌肉群优先,小肌肉群其次:先俯卧撑
多关节优先,少关节其次:先俯卧撑
前后两个运动不要使用相同肌肉群:避免疲劳造成损失
大强度在前小强度在后:先引体向上,后trx拉伸
## 模型搭配
