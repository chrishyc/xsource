# 心学核心
无善无恶是心之体,有善有恶是意之行,知善知恶是良知,为善去恶是格物
世界观,人生观,价值观,方法论
# 世界观
1.心有多大世界就有多大:万物一体->和->把天地万物当做自己身体的一部分
2.道理一直都在自己内心:世界是客观存在的,但是世界应该是什么样子,是由我的心来决定的。 表现在朋友上就是友谊,表现在父母上就是孝道,表现在亲人就是亲情
3.喜恶同因,善恶同体,世界无善无恶,跟随内心把握好程度:花草无善无恶,有的人喜欢花想除草,有的人喜欢草坪就要除花,有的人觉得花草应该自由生长.
4.无善无恶和佛家无欲无求区别:正确的理解世界,不要因为自己的好恶和客观世界对立，不要逆着客观大趋势去做事。理解世界的客观规律后，以做事的心态和提升自己能量的心态去合理处理自己和世界的关系

# 人生观
1.相信每个人都有可能性:人人平等,每个人都有自己的品格和能力,不能有凌驾于他人的傲慢,
2.思想自由独立才会自尊,自尊才会自信
3.内圣外王:遵从自己的内心做自己觉得合理的事情,自己可以反省自己，对外部不合理的事情有能力解决，把主动权掌握在自己手中。
4.因地制宜顺势而为:清楚自己的人生志向和方向,因地制宜顺势而为,不要逆势而行
5.有善有恶是环境因素导致

# 价值观
1.为什么有的人知善行恶:1.社会风气影响 2.欲望影响 
2.知行合一
