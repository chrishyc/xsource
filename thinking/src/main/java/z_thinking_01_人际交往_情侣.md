# 如何和对方深入交往？
## 和对方的原生家庭形成的创伤去做相反的事情
两个人在亲密关系中呈现出来的问题绝大数都是早期原生家庭所带来的，我们每个人身上都有创伤，这些创伤100%会开启，只有爱人才能走进我们内心深处
只有爱人才能达到我们跟父母的亲密程度，只有爱人才会有这么深的信任和依赖。爱人相处的方式就是我们和父母相处的方式，会把我们各自的创伤激发出来，
如果能从对方内心深入去弥补治愈这些需要就会很厉害

## 但凡他厌恶排斥的事情,我们都要朝着相反的方向去做
这种关系在普通朋友不容易表现出来,只有亲密关系才会表现出来。
比如父母的否定,我们要多给予肯定,赞美
父母比较严格，我们要多给予宽松的自由度，不要过于挑剔
父母经常忽视他，我们要多给予重视
父母贬低他，就要更多尊重
