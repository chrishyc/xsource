# 旅游体验分类
这周末在家回顾了一下自己这两年去过的地方，每次都尽量压低预算保证体验感不降低太多,发现自己的户外
第一诉求是**对生命力的探索和丰富多彩的风情体验**，具体可以初步划分为三种类型:

## 商团长线
和驴友商团出去走长线，主要考虑探险体验值，这种方式是最纯粹最原汁原味的徒步体验。
但是长线报团费用较高, 一般会权衡预算费用和体验感。目前只有风险较高的高原和无信号偏远地区，
才考虑长线报团

## 和朋友长线
和朋友出去走长线，需要权衡难度系数和体验感，但和朋友出行是最开心最难忘的方式，最值得投入成本的方式。
费用和体验天花板很高，依赖自己的组织调研能力， 目前最喜欢的方式， 找合适的队友较难，会找一些成熟度较高
风景还不错的路线和朋友一起

## 一个人长线
自己出去旅行，主要考虑丰富多彩的风情体验。自己走长线会觉得比较孤单乏味，看美景也不会特别激动，
出去徒步的核心诉求还是认识不同的人，满足自己的探索欲和冒险欲望， 自己出去更希望能和当地人互动，
所以会更倾向互动性较强的活动，会找一些有当地特色民俗活动参与感强的路线

## 商团短线
短线价格实惠,费用不是首要考虑,主要考虑风景和路线探索欲和人文风情。对短线的预期较低，风景好或者路线有探索欲或者
有人文风情，就会比较感兴趣。
